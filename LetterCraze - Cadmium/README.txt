Cadmium Readme
For best results please run the application on a PC.

Game
To run: Run LaunchLetterCraze.java as a Java App

Builder
To run: Run LaunchBuilder.java as a Java App
To Delete: Select a level from the select level menu and click delete
Don't delete or create a theme level, things will break.
To Create/Load: To create a custom level to play, you must make a new one and overwrite a previously
existing one, or load an existing one, edit, then save.
